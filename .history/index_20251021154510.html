Bienvenue sur ce jeu mémory. Venez tester votre mémoire en retrouvant les paires d'images identiques le plus rapidement possible ! Et faites le meilleur score !

Régles du Jeu :

  • Retourne deux cartes. Sil les images sont identiques, le jeu laisse les cartes visibles.

  • Si les images sont différentes, les cartes se retournent face cachées là où elles étaient.

  • La partie est terminée lorsque toutes les cartes ont été assemblees en paires.


Si vous voulez vous de votre score. Inscrivez-vous via le formulaire. Le jeu garde les 5 meilleures scores. Si vous voulez relancer une partie cliquez sur la barre d'espace.
